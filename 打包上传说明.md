## 部署说明

### 快速部署
1. 解压打包文件
2. 安装依赖: `npm install` (各目录分别执行)
3. 配置数据库连接
4. 启动服务:
   - 后端: `cd backend && npm run dev`
   - 前端: `cd fe && npm run dev`
   - MCP服务器: `cd MCPServer && npm run build && npm start`

### Docker部署
```bash
cd docker
./localRun.sh  # 本地开发环境
# 或
./deploy.sh    # 生产环境
```
