# 劳务人员签到系统使用说明

## 系统概述
基于低代码Excel数据管理系统构建的劳务人员签到签退系统，支持劳务人员的签到、签退和工作时间自动计算。

## 签到表信息
- **表名**: sign_in_sheet
- **哈希值**: `a7b672aabd61efd9f39668fc4fa179fc`
- **创建时间**: 2025-11-10

## 字段说明
| 字段名 | 类型 | 说明 | 验证规则 |
|--------|------|------|----------|
| 姓名 | 字符串 | 劳务人员姓名 | 必填 |
| 手机号 | 字符串 | 劳务人员手机号码 | 必填，手机号格式 |
| 所在公司 | 字符串 | 劳务公司名称 | 必填，下拉选项 |
| 签到时间 | 字符串 | 签到时间记录 | 系统自动记录 |
| 签退时间 | 字符串 | 签退时间记录 | 系统自动记录 |
| 实际工作时间 | 字符串 | 工作时间计算 | 系统自动计算 |

## 公司选项
- 汇博劳务公司
- 恒信劳务公司  
- 临时工

## 使用流程

### 1. 签到流程
1. 访问签到表单URL
2. 填写姓名、手机号
3. 选择所在公司
4. 点击"签到"按钮（系统自动记录当前时间）
5. 提交表单

### 2. 签退流程
1. 访问签到表单URL  
2. 填写姓名、手机号
3. 选择所在公司
4. 点击"签退"按钮（系统自动记录当前时间并计算工作时间）
5. 提交表单

### 3. 查看签到记录
1. 访问数据查询URL
2. 查看所有签到记录
3. 支持搜索和筛选

## 技术实现

### 时间自动记录
系统使用JavaScript在客户端自动记录当前时间：
- 签到时间：点击签到按钮时的当前时间
- 签退时间：点击签退按钮时的当前时间
- 实际工作时间：签退时间 - 签到时间

### 表单验证
- 姓名：必填
- 手机号：必填，符合手机号格式（1开头，11位数字）
- 所在公司：必填，从下拉选项中选择

## 系统特点
- **零代码修改**：基于现有系统功能实现
- **自动时间记录**：无需手动输入时间
- **工作时间计算**：自动计算实际工作时间
- **数据安全**：基于现有系统的数据管理
- **易于扩展**：可随时添加新字段或功能

## 注意事项
1. 确保系统时间准确
2. 签到和签退需在同一设备上完成
3. 工作时间计算基于系统时间
4. 支持同一人员多次签到签退

## 维护说明
如需修改字段配置，请访问表结构编辑URL进行配置。
